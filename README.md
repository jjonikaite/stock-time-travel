# Stock Time Travel Simulator

UI scaffold using Next.js App Router, TypeScript, Tailwind CSS, Recharts, Supabase and Twelve Data API.

Quick start:

```bash
npm install
# set env vars in .env.local:
# NEXT_PUBLIC_SUPABASE_URL=...
# NEXT_PUBLIC_SUPABASE_ANON_KEY=...
# TWELVE_DATA_API_KEY=...

npm run dev
```

Pages:

- /portfolio
- /trade
- /history

This is a UI skeleton. Implement logic and wire up Supabase/Twelve Data next.
